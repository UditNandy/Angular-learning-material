Decorators

  • Decorators are functions that modify the behaviour of class and their members.
  • They provide a way to apply metadeta to the classes and their members.

Types of decorators

  • Class decorator : They are applied to classes. Example : @Component, @NgModule, @Pipe, @Directive, @Injectable.
  • property decorator : They are applied to variabeles. Example @Input, @Output.
  • Method decoratros : Example : @HostBinding
  • Parameter decorators : These are used inside class constructors. Example : @Inject
