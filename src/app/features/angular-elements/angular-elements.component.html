Angular Elements

  • It is a feature of angular which helps in converting the normal angular component into a native web component.
  • Web Components : They are basically custom HTML elements which are part of DOM Apis which can be used anywhere.

Steps for creating a custom element in angular

  • Complete the application first.
  • Install angular-elements.
  • Inject an injector using dependency injection.
  • Add ngDoBpptstrap method in app.module.ts and inside it we need to create a custom element by using createCustomElement method from @angular/elements.
  • Pass AppComponent as the first parameter to createCustomElement and add an injector as 2nd parameter and assign the result to a variable.
  • Now we need to define our selector for the custom element.
  • We can do that using customElements.define method where the 1st parameter is the tag name and 2nd is the element that we created in the previous step.
  • Remove the bootstrap array from the metadata befor building.
  • Now we need to create a file called concat.js where we need to concat the js files that we will get after build.
  • Install concat package.
  • Create a new script called bundle in package.json and assign it the command ng build --output-hashing none && node concat.js
  • Now copy the scipt file and add it to any html file and use the app selector to display the element.
  • If there are any input prop then they are comverted as "myName" => "my-name" and we can get the events by adding event listners.
