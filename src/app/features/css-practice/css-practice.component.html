Units in css

  • %age : Takes the space relative to its parent by the amount specified.
  • vw (view width) : Takes the space relative to the screen not relative to parent.
  • vh (view height) : Same as vw but used for height.

Display

  • Block : It takes up the full width available and starts on a new line.Examples include div, p, h1, h6 and li.
  • Inline : It does not start on a new line and only takes up as much width as necessary. Examples include span, a, strong and em.
  • Inline block : It behaves like an inline element but can have block-level properties like width and height. It does not start on a new line and only takes up as much width as necessary.
  • None : The element is removed from the normal document flow and does not take up any space.
  • Flex : The element becomes a flex container and its children become flex items.
  • Grid : The element becomes a grid container and its children become grid items.
Note : We can only set the width of block elements not of inline elements.

Box Sizing

The box-sizing property in CSS is used to control how the width and height of an element are calculated.
  • Border Box : The height and width we specify includes padding and content.If width is 100 and height is 100 then any padding specified will be included withing the given dimensions.If the padding and border size go beyond the specific width and height then the size will increase.
  • Content Box : The height and width we specify excludes padding and content.If width is 100 and padding is 100 then actual width will be 300.

Positioning

  • Static : Positions to the normal flow. There is no effect of top, buttom, etc.
  • Absolute :
    • It is used to position an element relative to its closest positioned parent (i.e. a parent which is not static). If no positioned parent is found it will position relative to the first parent that is view port.
    • When an element is positioned absolutely, it is taken out of the normal document flow, meaning it does not affect the position of other elements.
    • If we don't specify any top, button, etc then the element will come out of the normal flow and other elements will think that no element is present in that space but it will not change its position.
    • So eventually it will get overlapped with its next element.
  • Relative :
    • We can alter the position relative to its normal position.
    • If we don't specify top, left, etc, then the element will present in the normal flow and it will not get effected.
  • Fixed :
    • Position an element relative to the viewport, which means it stays in the same position even when the page is scrolled.
    • When an element is set to fixed, it is taken out of the normal document flow, similar to absolute;
  • Sticky :
    • It behaves like relative until it reaches a specified position of top left,buttom,right and after that it behaves like fixed.
    • If we donot specify any top, left, right, left it will behave like relative.

Example

  • Box 1 : Absolute to its parent (container)
  • Box 2 : Relative to its original position

Transform

It applies a 2d or 3d transformation to an element.
  • Scale : Used to increase or decrease the size of a element in x, y and z directions. (px)
  • Skew :Used to skew(twist) an element along x and y axis.(degree)
  • Translate :Used to shift the element from its original position (px)
  • Rotate :Used to rotate the element from its original position (deg)

Transform Origin Property

  • Allows us to change the position of transform elements.
  • Supported values : left, center, right, length, %.
  • Tells us on which point does the transformation needs to happen (basically the index or axis).

Transition Property

  • It helps us to change the property values smoothly over a specified duration.
  • To create a transisition property we need two things.
    • CSS property we want to add an effect to.
    • Duration of the effect.
  • If duration is not specified the transition will have no effect as the default duration is 0.
Different Types of transitions are
  • transition-delay : Adds a delay before starting the transition.
  • transition-property :Defines on which specific property the transition needs to be applied, like background-color.
  • transition-duration : Defines the duration for transition.
  • transition-timing-funtion : It can have the following values.
    • ease (default) : Starts the event with slow start, then speeds up and then ends slowly.
    • linear : Specifies a timing effect with same speed.
    • ease-in : Specifies a timing effect with slow start.
    • ease-out : Specifies a timing effect with slow end.
    • ease-in-out : Specifies a timing effect with slow start and end.
    • cubic-bezier(n,n,n,n) : Lets us define our own function in a cubic-bezier function.
